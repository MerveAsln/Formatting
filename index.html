Semantics in HTML

👉 In Html, forexample, the <h1> element is a semantic element, which gives the tags it wraps around the tool or a top level heading on your page.

Here is a top level heading Lorem ipsum dolor, sit amet consectetur adipisicing elit. Omnis, quis dolores? Ea commodi quae atque cupiditate, fuga dicta at vitae ut ipsam eos delectus maiores minus dolorum quos labore sed?

👉 By default most Browser’s user agent stylesheet will style an <h1> with a large font size to make it look like a heading ( although you could style it to look like anything you wanted) On the other hand you could make any element look like a top level heading

Consider the following Lorem ipsum dolor, sit amet consectetur adipisicing elit. Omnis, quis dolores? Ea commodi quae atque cupiditate, fuga dicta at vitae ut ipsam eos delectus maiores minus dolorum quos labore sed?

⇨This will render it to look like top level heading but it has no semantics value, so it will not get any extra benefis as describe above. It is therefore a good idea to use to right html element for the right job.

⇨ Html should be coded to represent the data that will be populeted and not based on its default presentation styling. Presentation ( How it should look ) is the responsibility of CSS.

Some of the benefis from written semantic markup are as follows
  • Search engines will consider its contents as important keywords to influences the page’s search rankings ( see SEO )
  • Screen readers can use it as a signpost to help visually impaired users navigate a page
  • Finding block of meaningful code is a significantly easier than searching through endless divs with or without semantic or namespaced classes
  • Suggest to the developer the type of data that will be populated.
  • Semantic naming mirrors proper custom element/component naming

SEO

 ⭐ SEO ( Search Engine Optimization) is the process of making a website more visible in search results, also termed improving search ranking.
⭐ Search engines crawl the web following links frog page to page and index to content found. When you search, the search engine displays the indexed content. Crawlers follow rules. If you follow those rules closely when doing SEO for a website , you give the site the best chances of showing up among the first results increasing the traffic and possibly revenue ( for ecommerce and ads )
⭐ Search engines give some guidelines for SEO, but big search engines keep result ranking as a trade secret. SEO combines official search engine guideline, empirical knowledge and theoretical knowledge from science papers or patents.
